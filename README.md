## Цикломатска комплексност

Цикломатската комплексност за searchBookByTitle е 5, истата ја добив преку формулата P+1, каде што P е бројот на предикатни јазли. Во случајoв P=4, па цикломатската комплексност изнесува 5.

За да се исполни Every Statement критериумот, потребно е со минимален број на тест случаи да се поминат сите наредби во функцијата searchBookByTitle. Поради трите различни завршетоци на функцијата (throw exception, return results и return null), минималниот број на тест случаи е 3.

**test 1:** Овој тест случај ја покрива првата патека односно line 1 во табелата. Се праќа празен стринг ("") како аргумент во функцијата. Кодот ја извршува почетната проверка, го исполнува условот title.isEmpty() и веднаш фрла IllegalArgumentException. Со ова се покриваат наредбите за валидација на влезот на самиот почеток од функцијата.
 
**test 2:** Овој тест случај ја покрива втората патека односно line 2 во табелата. Се пребарува наслов на книга која постои во библиотеката и чиј статус е слободен (borrowed = false). Кодот ја поминува почетната проверка, ја иницијализира листата за резултати, влегува во for циклусот, го задоволува if условот за совпаѓање и ја додава книгата во листата. На крајот, успешно ја извршува наредбата return results.

**test 3:** Овој тест случај ја покрива третата патека односно line 3 во табелата. Се пребарува наслов на книга која е веќе изнајмена (borrowed = true) или воопшто ја нема. Кодот ја поминува почетната проверка, ја креира листата и го врти циклусот, но if условот внатре во циклусот паѓа (бидејќи !book.isBorrowed() е неточно). Листата останува празна, па кодот ја активира последната проверка if (results.isEmpty()) и ја извршува наредбата return null.
